## Story

**As a** user,
**I want** to estimate when I can retire based on my current financial trajectory,
**so that** I can understand the gap between my current savings rate and my retirement goal.

## Acceptance Criteria

1. Page `/retirement` with retirement estimation calculator
2. Input fields: target monthly income in retirement, expected annual return rate (default 8%), current total invested (editable)
3. FIRE calculator: shows years to retirement based on current savings rate and investment returns
4. Gap analysis card: difference between current monthly savings and required monthly savings to reach goal
5. Timeline visualization: Recharts area chart showing projected portfolio growth until retirement target
6. Scenario comparison: "If you save R$ X more per month, you retire Y years earlier"
7. Uses projection engine from Story 5.1 for baseline savings rate data
8. Responsive layout: form fields stack on mobile
9. Loading skeleton while calculations run

## Dev Notes

### FIRE Calculator Formula

```typescript
// FI Number = Annual Expenses * 25 (4% rule)
const fiNumber = targetMonthlyIncome * 12 * 25;

// Years to FI (with compound interest)
// FV = PV * (1+r)^n + PMT * ((1+r)^n - 1) / r
// Solve for n when FV = fiNumber
const monthlyRate = annualRate / 12;
const monthlySavings = avgIncome - avgExpenses;

// Iterative approach for accuracy
function yearsToFI(currentPortfolio: number, monthlySavings: number, monthlyRate: number, target: number): number {
  let balance = currentPortfolio;
  let months = 0;
  while (balance < target && months < 600) { // cap at 50 years
    balance = balance * (1 + monthlyRate) + monthlySavings;
    months++;
  }
  return months / 12;
}
```

### Gap Analysis

```typescript
// Required monthly savings to reach FI in targetYears
// PMT = (FV - PV * (1+r)^n) / (((1+r)^n - 1) / r)
const requiredSavings = calculateRequiredPMT(fiNumber, currentPortfolio, monthlyRate, targetYears * 12);
const gap = requiredSavings - currentMonthlySavings;
```

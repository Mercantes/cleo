## Story

**As a** free-tier user,
**I want** to see a clear paywall when I hit a limit and easily upgrade to Pro,
**so that** I understand what I'm missing and can upgrade with minimal friction.

## Acceptance Criteria

1. Paywall modal component shown when user hits a tier limit (transactions, chat, bank connections)
2. Paywall displays: feature name, current usage, limit, and what Pro unlocks
3. "Upgrade to Pro" CTA button in paywall that redirects to `/upgrade` page
4. `/upgrade` page with plan comparison table: Free vs Pro features side-by-side
5. Pricing display: monthly and annual options with annual discount highlighted
6. "Start Pro" button initiates Stripe Checkout session and redirects to Stripe
7. Success page after checkout: confetti animation, "Welcome to Pro!" message, redirect to dashboard
8. Paywall integrates with tier middleware 403 responses from API
9. Responsive: paywall modal is full-screen on mobile, centered dialog on desktop

## Dev Notes

### Paywall Trigger Pattern

```typescript
// In API call error handling:
try {
  const response = await fetch('/api/chat', { method: 'POST', body });
  if (response.status === 403) {
    const error = await response.json();
    if (error.error === 'TIER_LIMIT_REACHED') {
      showPaywall({ feature: error.feature, current: error.current, limit: error.limit });
      return;
    }
  }
} catch (e) { /* ... */ }
```

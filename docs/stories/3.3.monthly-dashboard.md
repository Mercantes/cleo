## Story

**As a** user,
**I want** to see a visual summary of my monthly finances,
**so that** I understand my financial situation in seconds.

## Acceptance Criteria

1. Page `/dashboard` with summary cards: total income, total expenses, month balance, savings rate
2. Bar chart: income vs expenses comparison for the last 6 months (Recharts)
3. Donut chart: expense breakdown by category (top 5 + "Outros") (Recharts)
4. Variation card: "You spent X% more/less than last month"
5. Subscriptions card: total active subscriptions and monthly committed amount
6. Real user data from imported transactions (not mocked)
7. Loading skeleton while data loads
8. Responsive: cards stack on mobile, grid on desktop
9. Month selector to view previous months

## Dev Notes

### Recharts Usage

```typescript
import { BarChart, Bar, XAxis, YAxis, Tooltip, ResponsiveContainer, PieChart, Pie, Cell } from 'recharts';

// Bar chart data format:
const monthlyData = [
  { month: 'Out', income: 5000, expenses: 3200 },
  { month: 'Nov', income: 5200, expenses: 3800 },
  // ...
];

// Pie chart data format:
const categoryData = [
  { name: 'Alimentação', value: 1200, color: '#EF4444' },
  { name: 'Transporte', value: 800, color: '#F59E0B' },
  // ...
];
```

### Month Format

- API param: `YYYY-MM` (e.g., `2026-03`)
- Display: Portuguese month name (e.g., "Março 2026")
- Navigation: arrow buttons or dropdown, limited to months with data

### Savings Rate Calculation

```typescript
const savingsRate = totalIncome > 0
  ? ((totalIncome - totalExpenses) / totalIncome * 100).toFixed(1)
  : 0;
```

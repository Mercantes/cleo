## Story

**As a** visitor,
**I want** to see a compelling landing page explaining Cleo's value proposition,
**so that** I understand the product and decide to sign up.

## Acceptance Criteria

1. Hero section with headline, subheadline, CTA "Começar grátis"
2. Value proposition section (3 pillars): dados automáticos, IA contextual, projeções
3. "Como funciona" section (3 steps): conecte, pergunte, planeje
4. Comparison section: Cleo vs planilhas vs outros apps
5. Trust signals: "Regulado pelo Banco Central", "Dados criptografados", "Não armazenamos senhas"
6. Pricing section: Free vs Pro tiers with features
7. Footer with institutional links
8. CTA buttons redirect to `/signup`
9. 100% responsive (mobile-first)
10. Meta tags SEO and Open Graph configured

## Dev Notes

### Route Structure

```
src/app/
├── (marketing)/              # Public pages group
│   ├── layout.tsx            # Marketing layout (header + footer)
│   └── page.tsx              # Landing page (all sections)
```

### Design Guidelines

- **Mobile-first:** Design for mobile, enhance for desktop
- **Server Components:** Entire landing page is a Server Component (no client JS)
- **Performance:** No heavy client-side JS, images optimized with next/image
- **Typography:** Use Geist font (already configured)
- **Colors:** Use Cleo brand tokens from globals.css (primary, success, danger)

### Content (Portuguese BR)

All text should be in Brazilian Portuguese. Key copy:
- CTA: "Começar grátis", "Saiba mais", "Criar conta"
- Trust: "Regulado pelo Banco Central", "Dados criptografados"
- Pricing: "Grátis", "Pro"

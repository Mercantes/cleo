## Story

**As a** user with imported transactions,
**I want** my transactions to be categorized automatically with confidence tracking,
**so that** I can see where my money goes without manual categorization.

## Acceptance Criteria

1. Enhance existing `src/lib/ai/categorize.ts` to add confidence scores per transaction
2. Add `category_confidence` field to transaction records (column already exists in DB schema)
3. Transactions with confidence < 0.70 flagged for review (optional future UI)
4. Token usage logged per categorization batch for cost monitoring
5. API route `GET /api/transactions/uncategorized` returns transactions without category
6. Fallback chain: AI → Pluggy raw_category → "Outros"
7. Re-categorization endpoint `POST /api/categorize` supports re-running on specific transactions

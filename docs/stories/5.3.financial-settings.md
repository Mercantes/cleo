## Story

**As a** user,
**I want** a settings page where I can edit my profile, manage connected banks, and configure notification preferences,
**so that** I can customize Cleo to my needs and control my connected accounts.

## Acceptance Criteria

1. Page `/settings` with tabbed navigation: Profile, Banks, Notifications
2. Profile tab: edit display name, preferred currency (BRL default), save with success toast
3. Banks tab: list all connected bank accounts with bank name, last sync date, connection status
4. Banks tab: disconnect button per connection with confirmation dialog
5. Banks tab: "Connect new bank" button that opens Pluggy Connect widget
6. Notifications tab: toggle switches for email notifications (weekly summary, unusual spending alerts, subscription reminders)
7. All changes persist to Supabase immediately with optimistic UI updates
8. Loading skeleton while settings load
9. Responsive layout: tabs become accordion on mobile

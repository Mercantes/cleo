## Story

**As a** product owner,
**I want** a freemium tier system that enforces limits for Free users and unlocks full access for Pro users,
**so that** the product can monetize while offering a useful free tier.

## Acceptance Criteria

1. Tier definitions: Free (100 transactions/month, 30 chat messages/month, 1 bank connection) and Pro (unlimited all)
2. Tier checking utility function `checkTierLimit(userId, feature)` returns `{ allowed: boolean, current: number, limit: number, tier: string }`
3. Enforcement middleware for API routes that checks tier limits before processing requests
4. User tier stored in profiles table (`tier` column: 'free' | 'pro', default 'free')
5. Usage tracking: increment counters in usage table per feature per billing period
6. Tier status component showing current usage vs limits in the UI
7. Graceful limit-reached response: return 403 with structured error including upgrade CTA data
8. Admin-safe: tier can be updated via Supabase dashboard or API


### Usage Tracking Table

```sql
CREATE TABLE usage_tracking (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID NOT NULL REFERENCES profiles(id) ON DELETE CASCADE,
  feature TEXT NOT NULL, -- 'transactions', 'chat', 'bank_connections'
  count INT NOT NULL DEFAULT 0,
  period TEXT NOT NULL,  -- '2026-03' for monthly features
  UNIQUE(user_id, feature, period)
);

-- RLS: users can only read their own usage
CREATE POLICY "Users read own usage" ON usage_tracking
  FOR SELECT USING (auth.uid() = user_id);
```

### Middleware Pattern

```typescript
export function withTierCheck(feature: TierFeature, handler: NextHandler) {
  return async (req: NextRequest) => {
    const userId = await getCurrentUserId(req);
    const check = await checkTierLimit(userId, feature);
    if (!check.allowed) {
      return NextResponse.json({
        error: 'TIER_LIMIT_REACHED',
        feature,
        current: check.current,
        limit: check.limit,
        upgradeUrl: '/upgrade',
      }, { status: 403 });
    }
    const response = await handler(req);
    await incrementUsage(userId, feature);
    return response;
  };
}
```

## Story

**As a** new user,
**I want** a guided step-by-step onboarding flow after signup,
**so that** I can connect my bank, categorize transactions, and set financial goals without feeling overwhelmed.

## Acceptance Criteria

1. Onboarding flow triggered after first login (check `onboarding_completed` flag in profiles)
2. Step 1 — Connect Bank: guided Pluggy Connect widget with explanation of why and how
3. Step 2 — Review Categories: show imported transactions with AI-suggested categories, allow corrections
4. Step 3 — Set Goals: set a monthly savings target and optional retirement goal
5. Progress indicator: visual stepper showing current step (1/3, 2/3, 3/3)
6. Skip option: "Skip for now" button on each step, skipped steps shown as reminders in dashboard
7. Completion celebration: animation + "You're all set!" message with redirect to dashboard
8. Onboarding state persisted: user can close browser and resume from last completed step
9. Responsive: full-screen flow on mobile, centered card on desktop

```typescript
type OnboardingStep = 'connect-bank' | 'review-categories' | 'set-goals' | 'complete';

interface OnboardingState {
  currentStep: OnboardingStep;
  stepsCompleted: OnboardingStep[];
  stepsSkipped: OnboardingStep[];
  isComplete: boolean;
}

// Step transitions:
// connect-bank → review-categories → set-goals → complete
// Any step can be skipped
```

```typescript
}
```
